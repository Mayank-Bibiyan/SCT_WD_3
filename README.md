# 🎮 Tic Tac Toe Game

A modern and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript** as part of my **Web Development Internship at SkillCraft Technology**.

## ✨ Features

* 👥 Friend Mode (Player vs Player)
* 🤖 AI Mode (Player vs Computer)
* 🎨 Attractive Modern UI
* 🌈 Different Colors for X and O
* 📱 Responsive Design
* 🔄 Restart Game Functionality
* 🏆 Winner Detection
* 🤝 Draw Detection
* ⚡ Smooth User Experience

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📷 Project Preview

The game provides two playing modes:

* Friend Mode for local multiplayer
* AI Mode to challenge the computer

Players take turns placing X and O on the board, and the game automatically detects wins and draws.

## 🎯 Learning Outcomes

Through this project, I gained practical experience in:

* DOM Manipulation
* Event Handling
* Game Logic Development
* Responsive Web Design
* JavaScript Fundamentals
* User Interface Design

## 🚀 How to Run

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser
